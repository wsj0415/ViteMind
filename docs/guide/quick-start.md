# 快速上手 ViteMind

欢迎来到 ViteMind！这是一个为您打造的 AI 情报与工具中心。以下是使用本站的简明指南。

## 🔍 浏览 AI 情报

1.  **访问首页**：点击左上角的 **ViteMind** 图标或导航栏的 **首页**。
2.  **搜索情报**：在首页顶部的搜索框中输入关键词（如 "OpenAI", "RAG"）。
3.  **标签筛选**：点击搜索框下方的标签（如 `#LLM`, `#SECURITY`）快速过滤相关新闻。
4.  **阅读详情**：点击任意新闻卡片，即可在弹出的模态框中阅读 AI 生成的深度摘要。

## 🛠️ 使用 AI 军火库

1.  **进入导航**：点击顶部导航栏的 **AI 导航**。
2.  **查找工具**：
    *   使用顶部的分类按钮（Coding, Image, Video 等）进行筛选。
    *   使用搜索框查找特定工具。
3.  **直达官网**：点击工具卡片右上角的箭头 `↗`，直接跳转至工具官方网站。

## 📧 订阅更新

1.  **订阅入口**：点击顶部导航栏的 **订阅**。
2.  **提交邮箱**：在输入框中填入您的邮箱地址并回车。
3.  **获取情报**：您将定期收到我们精选的 AI 行业动态（无垃圾邮件，仅限干货）。
